المعادلات الكيميائية

MnO₂ + 4HCl → MnCl₂ + Cl₂ + 2H₂O
Cl₂ + H₂O → HCl + HClO

تعليمات التجربة

  1. اسحب الأدوات إلى مساحة العمل
  2. أضف المواد الكيميائية
  3. صل الأنابيب بين الأواني
  4. اضغط "بدء المحاكاة"

استنتاج التجربة

في هذه التجربة قمنا بتحضير غاز الكلور من خلال تفاعل حمض الهيدروكلوريك (HCl) مع ثاني أكسيد المنجنيز (MnO₂).

المعادلة الكيميائية: MnO₂ + 4HCl → MnCl₂ + Cl₂ + 2H₂O

ثم مررنا غاز الكلور عبر الماء لتنقيته، ومن خلال حمض الكبريتيك لتجفيفه، وأخيراً تم تجميعه في الزجاجة المخصصة.

غاز الكلور الناتج هو غاز سام أصفر مخضر يستخدم في تطهير المياه وفي الصناعات الكيميائية.
